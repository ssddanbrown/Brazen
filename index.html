The Brazen ProjectAn Opinionated Website Starter Kit

Brazen is a simple html, gulp, sass/css project start kit. The elements have been inspired by bootstrap but has been designed to be more semantic and light-weight.

Brazen is designed to be used as a base rather than extending & overriding. Rather than being a css libraray of components, Brazen styles the base html elements and adds many usefull helper classes for rapid development without tempting you to use the same building blocks for everything you build.

Brazen is currently is heavy development
